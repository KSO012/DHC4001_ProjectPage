Introduction

고혈압[1]은 심혈관 질환과 같은 합병증의 주요 위험 요인 중 하나로, 조기 발견 및 관리가 중요합니다. 그러나 고혈압은 초기 증상이 뚜렷하지 않아, 정기적인 건강검진을 통한 모니터링을 필요로 합니다. 따라서, 본 프로젝트는 국민건강보험공단 건강검진정보 데이터를 활용하여 고혈압을 예측하는 모델을 개발하고자 하였습니다.

Method Summary

국민건강보험공단의 2019~2022년 건강검진 데이터를 사용하여 고혈압 예측 모델을 구현하였습니다. 데이터 전처리 과정에서 결측치를 처리하고, NIH[2] 기준에 따라 수축기·이완기 혈압 데이터를 통해 고혈압 여부 column을 추가하였습니다. 이후, Random Forest를 활용하여 주요 특성을 선별하고, 상관관계 분석을 통해 최종 특성들을 선택하였습니다. 마지막으로, Logistic Regression, Decision Tree, Random Forest, XGBoost 등으로 다양한 모델을 만들고 성능을 평가하였습니다.

Method

csv file processing

국민건강보험공단의 2019~2022년 건강검진 데이터 csv 파일을 확인한 결과, 2019년 데이터의 column 수와 명칭이 다른 연도(2020년~2022년)의 데이터와 상이한 것을 확인하였습니다. 이에 따라, 2019년 데이터의 컬럼 수와 이름을 다른 연도의 데이터와 일치하도록 수정하였습니다.

preprocessing

먼저, 결측치가 있는 행을 모두 제거하였습니다. 고혈압 예측 모델에 있어서, 결측치가 있는 행을 제외하는 것이 다른 값으로 대체하는 것보다 더 적절할 것으로 판단하였기 때문입니다.

다음으로, 고혈압 여부 column을 추가하였습니다. 고혈압 여부는 NIH 기준에 따라 수축기·이완기 혈압 데이터를 바탕으로 생성하였습니다.

이후, 고혈압 여부 column의 데이터가 불균형한 것을 확인하였습니다. 이를 해결하기 위핸 under-sampling 기법을 적용하여 고혈압이 아닌 환자와 고혈압인 환자의 데이터 균형을 맞추었습니다.

feature importance

본 프로젝트는 수축기·이완기 혈압을 제외한 다른 데이터로 고혈압을 예측하는 것을 목표로 합니다. 이에 기준년도와 가입자일련번호, 구강검진수검여부는 기여도 낮을 것으로 생각하였고, 기준년도, 가입자일련번호, 수축기혈압, 이완기혈압, 구강검진수검여부 column을 사전에 제외하였습니다.

feature importance는 예측을 수행할 때 각 입력 변수가 예측 결과에 미치는 상대적인 영향을 나타내며, 변수의 중요성을 순위로 나열하는 데에 사용됩니다. 따라서, 모델 학습에 반영할 feature들을 선택하기 위하여 Random Forest를 활용하였습니다.

또한, 기준으로 삼은 3개의 축(0.05, 0.06, 0.07)을 넘는 데이터 column과 고혈압 column의 correlation을 확인해보았습니다.

Model

Experimental Results

Logistic Regression 모델은 기준 축을 0.06으로 설정하였을 때, Decision Tree 모델은 기준 축을 0.05로 설정하였을 때, Random Forest 모델은 기준 축을 0.05로 설정하였을 때, XGBoost 모델은 기준 축을 0.05로 설정하였을 때, 가장 좋은 성능을 나타낸 것을 확인할 수 있습니다. 또한, 실험한 4개의 모델 중 가장 성능이 좋은 모델은 XGBoost 모델이라는 것을 알 수 있습니다.

Conclusion

본 프로젝트를 통해 국민건강보험공단 건강검진정보 데이터를 활용하여 고혈압을 예측하는 모델을 개발하였습니다. 4개년 치의 건강검진 데이터를 사용하였으며, 데이터 전처리 과정에서 일치하지 않는 column을 일치시키는 작업을 수행하였습니다. 수정한 데이터를 기반으로. 결측치를 처리하고, 수축기·이완기 혈압 데이터를 통해 고혈압 여부 column을 추가하였으며, under-sampling 기법으로 데이터 불균형을 해결하습니다 또한, 주요 특성을 선별하는 작업을 통해 모델에 쓰일 핵심 feature을 선별하였습니다. 이후, 선정된 feature 별로 4개의 모델(Logistic Regression, Decision Tree, Random Forest, XGBoost)의 예측 성능을 확인해 보았습니다.

XGBoost 모델이 가장 좋은 성능을 보인 이유는 약한 분류기(weak learner)를 보완하여 강력한 분류기(strong learner)를 만드는 방식이기 때문으로 판단되며, 모델 학습에 사용되는 feature는 개수보다는 기여도가 중요하다는 결론을 내릴 수 있었습니다.

향후 프로젝트에서는 고혈압 예측 모델의 성능을 높이기 위해 hyper-parameter 튜닝 기법을 적용하고자 합니다. 최적의 hyper-parameter를 찾아 모델 학습에 반영한다면, 모델의 성능을 향상시킬 수 있을 것으로 생각합니다.
